# LexiForge

LexiForge — это десктопный прототип приложения для изучения языков. Он построен на `PySide6` и `Qt WebEngine`, интерфейс на HTML/CSS/JavaScript. Пользователь может описать, что именно хочет выучить, после чего приложение генерирует словарные карточки с помощью OpenAI и запускает учебный сценарий на их основе.

## Текущие возможности

* Генерация словарных карточек с использованием OpenAI Responses API
* Учебный сценарий с разными типами заданий: объяснение, сопоставление, перевод, заполнение и вопросы
* JSON-план урока, который временно используется как сценарий урока

## Быстрый старт

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Glebhl/LexiForge.git
cd LexiForge
```

### 2. Создайте и активируйте виртуальное окружение

```bash
python -m venv venv
venv\Scripts\Activate.ps1
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Настройте переменные окружения

Создайте файл `.env` в корне проекта и добавьте ваш ключ OpenAI API:

```env
OPENAI_API_KEY=your_openai_api_key_here
```

Без `OPENAI_API_KEY` генерация карточек работать не будет.

### 5. Запустите приложение

```bash
python main.py
```

## Статус

Проект пока находится на ранней стадии. Генерация словарных карточек уже работает через OpenAI, но полноценный пайплайн уроков все еще частично опирается на временные JSON-данные.
